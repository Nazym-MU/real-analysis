If a series converges absolutely, then any rearrangement of this series converges to the same limit.

**Proof:**
Assume $\sum_{k=1}^\infty a_k$ converges absolutely to $A$ (see [[Absolute and conditional convergence of a series]]), and let $\sum_{k=1}^\infty b_k$ be a rearrangement of $\sum_{k=1}^\infty a_k$. Use $$s_n = \sum_{k=1}^n a_k = a_1 + a_2 + \cdots + a_n$$ for the partial sums of the original series and use $$t_m = \sum_{k=1}^m b_k = b_1 + b_2 + \cdots + b_m$$ for the partial sums of the rearranged series. Thus WTS $(t_m) \to A$.
Let $\epsilon > 0$. By hypothesis, $(s_n) \to A$, so choose $N_1$ s.t. $$|s_n - A| < \frac{\epsilon}{2}$$ for all $n \geq N_1$. Because the convergence is absolute, we can choose $N_2$ s.t.  $$\sum_{k=m+1}^n |a_k| < \frac{\epsilon}{2}$$ for all $n > m \geq N_2$. Take $N = \max\{N_1, N_2\}$. We know that the finite set of terms $\{a_1, a_2, a_3, \dots, a_N\}$ must all appear in the rearranged series. Thus, choose $M = \max\{f(k): 1 \leq k \leq N\}$.
Now, if $m \geq M$, then $(t_m - s_N)$ consists of a finite set of terms, the absolute values of which appear in the tail $\sum_{k=N+1}^\infty |a_k|$. Our choice of $N_2$ guarantees that $|t_m - s_N| < \epsilon/2$, and $$|t_m - A| = |t_m - s_N + s_N - A| \leq |t_m - s_N| + |s_N - A| < \frac{\epsilon}{2} + \frac{\epsilon}{2} = \epsilon$$ whenever $m\geq M. \qquad \qquad \square$
