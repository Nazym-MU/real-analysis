Let $\lim a_n = a$, and $\lim b_n = b$. Then, 
1. $\lim(ca_n) = ca \space \forall \space c \in \mathbb R$
2. $\lim(a_n + b_n) = a + b$
3. $\lim(a_n b_n) = a b$
4. $\lim(a_n / b_n) = a / b$, provided $b\neq 0$

**Proof:**
1. Consider $c\neq 0$. Follow the template. Let $\epsilon > 0$ be arbitrary. WTS: $$|ca_n - ca| < \epsilon.$$ Now, $|ca_n - ca| = |c||a_n - a|$. We are given $(a_n) \to a$. Choose $N$ s.t. $|a_n-a| < \frac{\epsilon}{|c|}$ whenever $n \geq N$. Observe that $\forall n \geq N$, $$|ca_n-ca|=|c||a_n-a|<|c|\frac{\epsilon}{|c|}=\epsilon$$
2. WTS $|(a_n+b_n) - (a+b)|$ < $\epsilon$. Via Triangle inequality $$|(a_n + b_n) - (a+b)| = |(a_n-a) + (b_n-b)| \leq |a_n-a| + |b_n-b|.$$ Let $\epsilon > 0$ be arbitrary. Using the hypothesis that $(a_n) \to a$, we know there exists an $N_1$ s.t. $$|a_n-a| < \frac{\epsilon}{2} \qquad \text{whenever} \qquad n \geq N_1.$$ Using the assumption that $(b_n) \to b$ means that we can choose an $N_2$ s.t. $$|b_n-b| < \frac{\epsilon}{2} \qquad \text{whenever} \qquad n \geq N_2.$$ Choose $N = \max \{N_1, N_2\}$ to ensure that if $n \geq N$, then $n \geq N_1$ and $n \geq N_2$. Therefore,  $$|(a_n+b_n) - (a+b)| \leq |a_n-a| + |b_n-b| < \frac{\epsilon}{2} + \frac{\epsilon}{2} = \epsilon \qquad \forall n \geq N$$
3. WTS $|a_nb_n - ab| < \epsilon$. $$|a_nb_n-ab| = |a_nb_n - ab_n + ab_n - ab| \leq$$ $$ \leq |a_nb_n - ab_n| + |ab_n - ab| = $$$$= |b_n||a_n-a| + |a||b_n - b|$$ Let $\epsilon > 0$ be arbitrary. We know that each component is $<\frac{\epsilon}{2}$ as in part 2. If $a \neq 0$, choose $N_1$ s.t.$$n \geq N_1 \qquad \text{implies} \qquad |b_n-b| < \frac{1}{|a|}\frac{\epsilon}{2}$$ In case of the left term, choose the worst-case estimate. Via [[Theorem 2.3.2]], there a bound $M>0$ s.t. $|b_n| \leq M$ $\forall \ n \in \mathbb N$. Choose $N_2$ s.t. $$|a_n-a| < \frac{1}{M} \frac{\epsilon}{2} \qquad \text{whenever} \qquad n \geq N_2.$$ Pick $N = \max\{N_1, N_2\}$. If $n \geq N$, then $$|a_nb_n - ab| \leq M|a_n-a| + |a||b_n-b| < M\left(\frac{\epsilon}{M2}\right) + |a|\left(\frac{\epsilon}{|a|2}\right) = \epsilon$$
4. WTS:$$(b_n) \to b \qquad \text{implies} \qquad \left(\frac{1}{b_n}\right) \to \frac{1}{b}$$ Choose the worst-case estimate on the size of $1/(|b||b_n|)$. An inequality $|b_n| \geq \delta > 0$ will lead to a bound on the size of $1/(|b||b_n|)$. Look far enough out into the sequence $(b_n)$ so that the terms are closer to $b$ than they are to 0. Consider $\epsilon_0 = |b|/2$. Because $(b_n) \to b$, there exists an $N_1$ such that $|b_n - b| < |b|/2$ for all $n \geq N_1$. This implies $|b_n| > |b|/2$. Next, choose $N_2$ such that $n \geq N_2$ implies $$|b_n-b| < \frac{\epsilon|b|^2}{2}$$ Let $N = \max\{N_1, N_2\}$, then $n \geq N$ implies $$\left|\frac{1}{b_n} - \frac{1}{b}\right| = |b-b_n|\frac{1}{|b||b_n|} < \frac{\epsilon|b|^2}{2}\frac{1}{|b|\frac{|b|}{2}} = \epsilon. \qquad \square$$
