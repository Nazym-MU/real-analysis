> There is no rational number whose square is 2.

Assume that there is no common factor. $p$ and $q$ are integers.
Proof by contradiction:
$$\left(\frac{p}{q} \right)^2 = 2$$
$$p^2 = 2q^2$$

$p^2$ is an even number $\to$ $p$ is an even number because square of an odd number is odd. 
Thus, $p = 2r$, where $r$ is an integer.
$$4r^2 = 2q^2$$
$$2r^2 = q^2$$
$q^2$ is an even number $\to$ $q$ is even
$p$ and $q$ are both even, given the original assumption to have no common factor.
$\to \left(\frac{p}{q} \right)^2 = 2$ cannot hold. $\square$


This is important because Greeks believed that given two line segments AB and CD, it would always be possible to find a third line segment whose length divides evenly into the first two. In modern terminology, this is equivalent to asserting that the length of CD is a rational multiple of the length of AB.
