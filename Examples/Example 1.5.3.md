If we set $E = \{2, 4, 6, ... \}$ be the set of even natural numbers, then we can show $N \sim E$. Let $f: N \to E$ be given by $f(n) = 2n$. From cardinality point of view, these two sets are equivalent. Saying E is smaller represents a point of view that is biased from the overexposure to finite sets.

Same for $N \sim Z$:
$$f(n) = \begin{cases} (n-1)/2 && \text{if } n \text{ is odd} \\ -n/2 && \text{if } n \text{ is even} \end{cases}$$
Even numbers become negative numbers in $Z$, and odd numbers become positive numbers and include 0.
