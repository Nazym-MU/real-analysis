$$C = \bigcap_{n=0}^\infty C_n$$
where $C_n$ is a set that is the union of two closed intervals, each with a length of $1/3^n$.

- The middle third is removed from every set.
- There are $2^n$ closed intervals in total.
- It has zero length (measures the same size as a single point).
- Uncountable. The cardinality is equal to the cardinality of $\mathbb R$.
- Dimension of the Cantor set is $\log 2/\log 3$.
	- Magnifying the Cantor set by a factor of 3 yields two copies of the original set.
	- Thus, if $x$ is the dimension, then it should satisfy $2=3^x$.
