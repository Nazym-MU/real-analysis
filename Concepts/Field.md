Any set where addition and multiplication are well-defined operations that are commutative, associative, and obey distributive property $a(b+c) = ab + ac$
There must be an additive identity, and every element must have an additive inverse.
There must be a multiplicative identity, and every nonzero element must have a multiplicative inverse

**Examples:**
- $Q = \{ \text{all fractions } \frac{p}{q} \text{ where } \text{ and } q \text{ are integers with } q \neq 0 \}$
	- has a natural order: given any two numbers $r$ and $s$, exactly one is true: $r > s, r = s, r < s$
- $Z$ or $N$ are not fields
- The finite set {0, 1, 2, 3, 4} is a field when addition and multiplication are computed mod 5
