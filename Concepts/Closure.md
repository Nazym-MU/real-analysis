Given a set $A ⊆ \mathbb R$, let $L$ be the set of all limit points of $A$. The closure of $A$ is defined to be $A= A\cup L$.

### Example
- if $A= {1/n : n ∈ \mathbb N}$, then the closure of $A$ is $\bar A = A\cup\{0\}$ 
- $\bar{\mathbb Q} = \mathbb R$ 
- If $A$ is an open interval $(a, b)$, then $\bar{A} = [a, b]$.
- If $A$ is a closed interval $[a, b]$, then $\bar{A} = A$.
