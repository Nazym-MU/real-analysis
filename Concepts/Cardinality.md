Refers to the size of a set

The set $A$ has the same cardinality as $B$ if there exists $f: A \to B$ that is 1-1 and onto. In this case, we write $A \sim B$

$Q$ is countable, and $R$ is not. But $R = Q \cup I$ (via [[Theorem 1.5.8]]), so $I$ must be uncountable, or otherwise $R$ would be countable. We conclude that $I$ is a much larger set than $Q$.
