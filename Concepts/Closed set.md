A set $F\in \mathbb R$ is closed if it contains its limit points.

### Example
A closed interval $[c, d] = \{x \in \mathbb R: c \leq x \leq d \}$ is a closed set. If $x$ is a limit point, then by [[Theorem 3.2.5]] there exists $(x_n) \subseteq [c,d]$ with $(x_n) \to x$. WTS: $x \in [c,d]$.

Using [[Theorem 2.3.4 - Order Limit Theorem]], because $c \leq x_n \leq d$, it follows that $c ≤ x ≤ d$ as well.
$\therefore \ [c, d]$ is closed
