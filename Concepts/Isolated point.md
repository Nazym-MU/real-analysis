A point $a\in A$ is an isolated point of $A$ if it is not a limit point of $A$.

#### Example:
$$A = \left\{ \frac{1}{n}: n\in \mathbb N \right\}$$

Given $1/n \in A$, choose $\epsilon = 1/n - 1/(n+1)$. Then, $$V_\epsilon(1/n) \cap A = \left\{ \frac{1}{n} \right\}$$
From the definition, it is not a limit point. The limit $0$ is not contained => $A$ is not closed.
