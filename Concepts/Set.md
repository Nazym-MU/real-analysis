Any collection of objects, e.g. a set of real numbers

**Notation:** $x \in A$, $x \notin A$

A set $A$ is countable if $N \sim A$. An infinite set that is not countable is called an infinite set.

### Examples
- $S = \{ r \in Q: r^2 < 2 \}$
	- Let S be a set of all rational numbers whose squares are less than 2
- $\varnothing$  - empty set
- Infinite sets
	$$\bigcup_{n=1}^\infty A_n \qquad \bigcup_{n\in N} A_n \qquad \text{or} \qquad A_1\cup A_2 \cup A_3 \cup \ldots$$
- Complement of A: $A^c = \{ x \in R: x \notin A \}$
