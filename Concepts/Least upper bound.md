A real number $s$ is the least upper bound (or supremum) for a set $A \subseteq R$ if it meets the two criteria:
1. $s$ is an upper bound for $A$
2. if $b$ is any upper bound for $A$, then $s \leq b$
	1. Alternatively: [[Lemma 1.3.8]]


Notation:
$s = \sup A$

$\sup(c+A) = c + \sup A$: [[Example 1.3.7]]
