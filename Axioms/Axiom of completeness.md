- Every nonempty set of real numbers that is bounded above has a least upper bound
	- A set $A\subseteq \mathbb{R}$ is bounded above if $\exists \space b \in R$ s.t. $a \leq b \space \forall \space a \in A$
	- [[Least upper bound]]
- Every nonempty set of real numbers that is bounded below has a greatest lower bound
	- A set $A\subseteq \mathbb{R}$ is bounded below if $\exists \space b \in R$ s.t. $l \leq a \space \forall \space l \in A$
	- [[Greatest lower bound]]
